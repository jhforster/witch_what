If you're wondering what this all means -- think of it as a sort of chatroom or "digital townsquare" for local residents.

It is set up to use Slack, which is a super popular app that is totally free (be sure to install the mobile app!). Basically, this is a quick experiment to see if this is a helpful format to help people connect with one another and discover all the fun stuff happening in Salem.
